# TREFIND

## 1. Cloning the repository
```
git clone https://github.com/fabianMM04/trefind.git
```
## 2. Install dependecies
```
cd trefind
npm install
```
## 3. Run project
```
npm start 
```
## 4. Go to
```
http://localhost:4200/
```
